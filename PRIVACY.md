# PixBuddy Privacy Policy

Last Updated: 2026

PixBuddy respects user privacy and collects minimal data necessary for functionality.

## Information Collected

PixBuddy may collect:

* Server IDs
* Channel IDs
* User IDs
* Command usage data

This data is used only to provide bot functionality.

## Data Usage

Collected data is used for:

* Command execution
* Moderation features
* Server configuration
* Bot improvements

## Data Sharing

PixBuddy does not:

* Sell user data
* Share data with third parties
* Store sensitive personal information

## Data Retention

Data is stored only as long as necessary for bot functionality and may be deleted at any time.

## Security

Reasonable measures are taken to protect data and prevent unauthorized access.

## Changes

This Privacy Policy may be updated at any time.
